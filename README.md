# Narrative Worlds VR — LOTR Project

A VR experience set in Tolkien's Middle-earth, built with Unity and the XR Interaction Toolkit.

### Book System

**Scripts** (in `Assets/Scripts/BookSystem/`):

| Script                 | Purpose                                                                                                                                           |
| ---------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------- |
| `VRBookController.cs`  | Renders story text pages through a TextMesh Pro field and provides next and previous page navigation mechanisms.                                  |
| `SceneBookSpawner.cs`  | Spawns the virtual book when an environment loads, injects the correct chapter text from `NarrativeChapters.cs`, and links interaction callbacks. |
| `NarrativeChapters.cs` | Static data container storing the text for each specific environment/chapter.                                                                     |

**Prefabs**:

- `VRBook` — Interactible book object containing the `VRBookController` and XRI grab interactables.

### Audio System

**Scripts** (in `Assets/Scripts/Audio/`):

| Script                   | Purpose                                                                                                  |
| ------------------------ | -------------------------------------------------------------------------------------------------------- |
| `AmbientSoundSpawner.cs` | Randomly fires spatialized 3D sounds around the player for a configurable duration on scene start.       |
| `AudiobookPlayer.cs`     | Loads audiobook files from `StreamingAssets/Audiobook/`, handles playback, chapters, and user bookmarks. |
| `AudiobookUI.cs`         | Drives the World Space Canvas UI for the audiobook player.                                               |

**Prefabs**:

- `AmbientSoundSystem` — Drag into any scene and assign audio clips in the Inspector.
- `AudiobookPlayer` — Fixed World Space Canvas panel.

**Audiobook File Setup**:

Place audio files in `StreamingAssets/Audiobook/`. Each file can optionally have companion files:

- `mybook.chapters.txt` : Authored chapters formatted as CSV (read by the app).
- `mybook.bookmarks.txt` : User bookmarks saved persistently at specific timestamps (auto-generated by the app).

### Scene Loading and Shared Systems

**Scripts**:

| Script                | Purpose                                                                                                                                                                              |
| --------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `LobbyUIManager.cs`   | Manages the title screen, environment selection menus, and scene transitions. Handles screen fade-to-black masks and audio crossfading between the lobby and the environment levels. |
| `PauseMenuManager.cs` | Intercepts the XRI menu button input to toggle a system menu offering a "Return to Lobby" function and master volume control.                                                        |

**Scene Flow**:

The user begins in the Lobby scene, which also acts as the baseline version of The Shire. Selecting The Shire from the UI dismisses the menus and immediately fades into the environment on the spot. Selecting Gollum's Cave or Mordor triggers a screen fade and an asynchronous Unity scene load to transfer the player.

### Project Setup

- Unity version: 6000.2.2f1
- XR Plugin: OpenXR / XR Interaction Toolkit
- TextMesh Pro required
- No additional packages needed for the audio or book systems
